This repository contains a simple GitHub Pages site for Sweet Jesus School Complex (Sweet Jesus School Complex — Accra).

Files added:
- docs/index.html — homepage
- docs/styles.css — basic styles
- README.md — this file

To publish the site using GitHub Pages:
1. Go to the repository Settings → Pages.
2. Under "Source", choose the branch: "main" and the folder: "/docs". Save.
3. Wait a minute; GitHub will show the site URL on that page (it will be something like: https://nanayaw356t-prog.github.io/Dr.lifeit/).
